Ropert Age Calculator (RACv1)

This program takes an age, and calculates it into Roperts using the Prime Ropert Scaling (PRS) value.
This project is released under GPL-3.0-only.

I thought there was almost a gap in free, open-source tools to calculate a user's age into Roperts, so I took a leap, and with a few tutorials and much trial + error: the Ropert Age Calculator v1.0.0 (GPL-3.0-only) was born.

The decision to release this project under the GNU General Public License 3.0 was a tough one, but I want there to be the freedom in anyone to improve, review, and reimpliment the code from my program. 

Thank you very much everyone!
